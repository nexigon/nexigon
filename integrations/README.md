# Nexigon Integrations

Ready-made integrations that extend Nexigon Agent with device management capabilities. Each integration is a self-contained directory of scripts, command definitions, and systemd units.

These integrations compose Nexigon's [primitives](https://nexigon.cloud/blog/2026-02-19-primitives-first-architecture/) (properties, events, repositories, commands) with system-specific tools. They are designed to be readable, modifiable, and replaceable.
